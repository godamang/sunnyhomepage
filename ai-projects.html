AI를 사용하는 아이에서 직접 설계하는 아이로

AI가 코드를 빠르게 만들어도 데이터와 논리가 맞는지 판단하려면 기본기가 필요합니다. 써니코딩은 먼저 읽고 생각하고 구현하는 힘을 기른 뒤 AI를 활용하고 검증하는 프로젝트로 확장합니다.

결과물보다 설계·수정·설명 과정

로봇·아두이노

센서 입력과 장치의 동작 조건을 연결해 온도 체크 손 소독 로봇과 압력 보호장비 같은 프로젝트를 구현합니다.

데이터·AI

데이터를 관찰하고 규칙을 찾으며 입력 데이터와 결과가 타당한지 확인합니다.

파이썬 게임

게임의 규칙과 상태 변화를 설계하고 오류를 수정하면서 텍스트 코딩 경험을 쌓습니다.

AI 결과 검증

코드가 실행된다는 사실만 보지 않고 누락된 조건, 잘못된 데이터와 비효율적인 흐름을 확인합니다.

프로젝트 수업의 네 단계

  1. 문제 발견 — 누구의 어떤 문제를 해결할지 정의합니다.
  2. 기능 설계 — 입력·처리·출력과 필요한 데이터를 나눕니다.
  3. 구현과 테스트 — 직접 만들고 예상과 다른 결과를 수정합니다.
  4. 발표와 검증 — 사용 기술, 실패와 개선 과정을 설명합니다.
